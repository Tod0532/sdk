 sh -ci "$(curl -fsSL https://smartcontracts.org/install.sh)"
2，创建新项目
dfx new mysite --no--frontend
3,在dfx.json文件编辑填加网页内容
3，项目运行
dfx start --background 
4，项目部署
dfx deploy
